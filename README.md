# TropicalBlindspots
Code and select data files used to generate the results in the manuscript Blind Spots in our Understanding of the Tropical Nitrogen Cycle.
Please contact Kate Nelson at kate.nelson@mail.mcgill.ca as needed. 

"TropicalBlindSpotsCode.Rmd" is the primary code document for this project, written in RMarkdown. BNF and N loss TBM outputs were obtained from Kou_Giesbrecht et al., and their processing is outlined here as well. "PFTs.Rmd" delineates how data from Sitch et al., were processed on a TBM-specific basis and requires accessing their data repository.
